# DraftStats Prototype

Small vanilla JS prototype for the `Track` flow:

- mobile-first date selection
- existing-event hints on the selected date
- existing-event reselection
- background creation of a new event when `date + set + index` is unique
- duplicate warning when the combination already exists
- in-memory only, no persistence yet

## Run

Open [index.html](C:\Users\steph\code\DraftStats\index.html) directly in a browser.

For the smoothest local dev loop, you can also serve the folder with any static file server.
